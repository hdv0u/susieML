# susieML
does uhh detect Susie

## Overview
susieML is a mini-tool that lets you train your own image detector model WITH your own images!

## Features
- CPU and GPU support(cuda)
- Binary-class detection
- Multi-class detection(WIP)
- A UI(very experimental)
- A save/load feature

## Disclaimer and Notes
This mini-tool is purely for fun and this is NOT affiliated with dsand00 at all.
Training/Inference speed varies on the local host machine.
